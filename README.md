Building ML Projects That Actually Work 
Hey everyone! I just wrapped up my Machine Learning internship with CodSoft and honestly — it was a solid learning experience. Built 5 real projects from scratch, each solving a different real-world problem.
Here's what I made:

Can an AI guess a movie's genre just from its plot?

Turns out yes — trained a classifier on thousands of movie descriptions and it picks up genre patterns surprisingly well. Try it yourself and see if it gets yours right.

→ Movie Genre Classifier - https://codsoft-nx9yhn3565ajwsinaktr7w.streamlit.app/

Spotting fraud before it happens

Credit card fraud detection is a classic ML challenge because fraud cases are rare — less than 1% of transactions. Getting the model to catch them without flagging everything is the tricky part. This one does it pretty well.

→ Fraud Detection App - https://securetransactionai.streamlit.app/

Who's about to leave?

Built a churn predictor for a telecom company dataset. The model figures out which customers are likely to cancel before they actually do — which is genuinely useful for any subscription business.

→ Churn Prediction App - https://thecustomerpulse.streamlit.app/

Spam or legit?

You know those "Congratulations! You've won!" texts? This classifier catches them instantly. Simple NLP but surprisingly satisfying to see it work.

→ SMS Spam Detector - https://messageshield.streamlit.app/

Teaching a model to write

This one was the most fun — a character-level model trained on Shakespeare, science texts, and poetry. You pick the style and temperature, and it generates new text. It's weird and creative in the best way.

→ Text Generator - https://inkscribe.streamlit.app/
